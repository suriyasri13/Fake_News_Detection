# 🚀 Deploying FactGuard AI Ultra

Follow these steps to host your project online for **FREE** so you can share it with anyone!

## 1. Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in.
2. Create a new repository named `fake-news-detection`.
3. Upload all the files from this folder to the repository.

## 2. Set Up Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io).
2. Connect your GitHub account.
3. Click **"New App"**.
4. Select your `fake-news-detection` repository.
5. Set the Main File Path to: `app.py`.
6. Click **"Deploy"**.

## 3. Share Your Link!
Once the deployment finishes (usually 2-3 minutes), you will get a professional link like:
`https://factguard-ai.streamlit.app/`
